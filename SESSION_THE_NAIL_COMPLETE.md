## How It Works: The Mathematics

Each domain positioned by its statistical properties:
- **X-axis:** Effect size (r-value) - stronger correlations further right
- **Y-axis:** Statistical significance (-log p-value) - more certain higher up
- **Z-depth:** Sample size (log scale) - more data = more opacity
- **Color:** Domain type (social=warm, psychological=cool)
- **Glow:** Certainty made visible as light

The perspective transformation at 30° pulls vertical elements toward the y-axis and horizontal elements toward the x-axis, forming the cross. Not forced - the mathematics permits it. The nail was always there in the data, waiting to be seen from the right angle.
