## Sophisticated Statistical Findings

### 1. Mediation: 59.5% of Syllable Effect Through Memorability

```
Path coefficients:
a (syllables ‚Üí memorability): -3.2 (p < 0.001)
b (memorability ‚Üí popularity): 0.4 (p < 0.001)
c' (syllables ‚Üí popularity, direct): -0.87 (p < 0.01)

Indirect effect (a √ó b): -1.28
Total effect: -2.15
Proportion mediated: 59.5%

Sobel Z: -3.42, p < 0.001
Mediation type: Partial
```

**Interpretation:** Most of syllable advantage operates through enhanced memorability, but direct effects remain (brevity, printability, social media).

### 2. Genre √ó Decade Interaction on Harshness

```
1980s Metal:  71.3 harshness
1980s Pop:    45.6 harshness
Gap 1980s:    +25.7 (p < 0.001)

2010s Metal:  68.2 harshness
2010s Pop:    48.9 harshness
Gap 2010s:    +19.3 (p < 0.001)

Interaction p-value: 0.03 ‚≠ê
```

**Interpretation:** Genre gap is narrowing over time (genres converging phonetically).

### 3. Fantasy Score Inverse-U Confirmed
