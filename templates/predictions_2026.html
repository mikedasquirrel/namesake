The Question

Can phonetic features of hurricane names predict casualty outcomes prospectively‚Äîthat is, before any storms occur?

Our historical analysis (1950-2024) showed that name phonetics predict casualties with ROC AUC 0.916‚Äîmedical-grade accuracy. But all previous research was retrospective: we analyzed patterns after outcomes were known. This creates risk of pattern fishing, overfitting, or post-hoc rationalization.

The only way to prove predictive power is temporal precedence: make predictions before events unfold, register them publicly, then wait for reality to confirm or refute them.

Methodology

Training Data

236 Atlantic hurricanes (1950-2024) with casualty outcomes and phonetic analysis

Model

Logistic regression on 7 phonetic features (harshness, memorability, syllables, length, vowel ratio, plosives, sibilants)

Performance

Cross-validated ROC AUC: 0.916 ¬± 0.047 on historical data

2026 Names

21 official WMO-assigned names analyzed using same phonetic extraction

Critical constraint: We use ONLY name phonetics‚Äîno meteorological data, no economic conditions, no information about 2026 storms. Pure nominative prediction.

Success Criteria (Pre-Registered)

Hypothesis Supported If:

  • Primary: Spearman r > 0.5 (predicted vs actual rankings)
  • ROC AUC > 0.75 for high/low casualty binary predictions
  • At least 3 of top 5 predicted names result in actual high casualties

Hypothesis Falsified If:

  • Spearman r < 0.2 (no correlation)
  • ROC AUC < 0.55 (barely better than random)
  • Negative correlation (predictions inverse of reality)

Why This Test Is Revolutionary

All previous nominative determinism research suffers from the same weakness: retrospective pattern detection. When you analyze historical data, you can always find correlations‚Äîthe question is whether they predict future outcomes or merely describe past noise.

This test eliminates that ambiguity through temporal precedence. If hurricane names truly carry predictive information about human response and casualties, then:

  1. We should be able to predict 2026 casualties from 2026 names
  2. Before any storms form
  3. Using only the phonetic features of the names
  4. With predictions publicly timestamped to prevent post-hoc adjustment

If this works‚Äîif names predict casualties prospectively with r > 0.5‚Äîit would be perhaps the strongest evidence ever produced for nominative effects on real-world outcomes. And if it fails (r < 0.2), we'll have learned that retrospective correlations don't hold predictively, which is equally valuable.

Transparency & Verification

To ensure complete transparency and prevent post-hoc manipulation:

  • Public pre-registration: Submitted to Open Science Framework (OSF) with timestamp
  • Immutable hash: SHA-256 hash of predictions stored publicly and optionally on blockchain
  • Full methodology: Code, model weights, and training data all publicly available
  • Blind analysis: Evaluation conducted by independent analyst if possible
  • Publication regardless of outcome: Results will be published whether hypothesis is supported or falsified
