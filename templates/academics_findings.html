The Seven Research Hypotheses

H1
The Authority Gradient Hypothesis
Prediction: Phonetic authority scores correlate with university prestige tier (Top-20 vs regional).
Example: "Worthington" (authority: 87) more common at Princeton than "Miller" (authority: 42) at regional universities. Target Effect: Cohen's d = 0.35-0.45
H2
The Syllable-Citation Paradox
Prediction: Memorability (short, simple names) predicts citation counts MORE than authority.
Mechanism: Citation generation is partially a function of name memorability. "Chen" easier to recall/cite than "Wojciechowski." Target: r = 0.18-0.24 (memorability ‚Üí h-index)
H3
Field-Specific Phonetic Signatures
Prediction: STEM faculty have harsher names than humanities faculty.
Rationale: STEM = precision, certainty ‚Üí harsh consonants. Humanities = nuance, interpretation ‚Üí softer phonetics. Target: d = 0.25-0.35 (STEM vs Humanities harshness)
H4
The Tenure Track Name Advantage
Prediction: Full professors have +12% higher authority scores than assistant professors at the same university.
Test: Control for university prestige, compare phonetics across ranks. If true, nominative determinism operates WITHIN institutions.
H5
The Top-20 Prediction Challenge (ROC AUC > 0.85)
Prediction: Machine learning model using ONLY name phonetics can predict Top-20 university employment with medical-grade accuracy.
If achieved: Would prove nominative determinism as powerful as traditional credentials. Paradigm-shifting result.
H6
Ethnic Name Penalty Quantification
Prediction: Ethnic names face -15 to -20 point prestige penalty controlling for credentials.
Test: Compare matched pairs (same credentials, different ethnic signals). Critical for identifying discrimination vs phonetics.
H7
The Publishing Name Effect
Prediction: Professors with memorable names have 18-25% more citations controlling for paper quality.
Mechanism: Memorable names ‚Üí better recall ‚Üí more citations. "Smith" cited more than "Szczepanski" for equivalent work.

What Statistical Success Looks Like

ROC AUC > 0.85 (Top-20 Prediction)
Everyday equivalent: If we achieve this, predicting Harvard professors from names would be as accurate as airport security screening or a credit score predicting loan default. Not perfect, but surprisingly reliable‚Äîlike correctly guessing someone's hometown from their accent 8-9 times out of 10. This would be paradigm-shifting.
Cohen's d = 0.35-0.45 (Authority Gradient)
Translation: Princeton professors' names sound ~35-45% more authoritative than community college professors. About the same difference as height between NBA players and average adults (6'7" vs 5'9"). Small but consistent and visible.
r = 0.18-0.24 (Memorability ‚Üí Citations)
Real-world impact: Name memorability explains 3-6% of citation variance. Sounds small, but consider: This is the difference between 100 citations vs 130 citations for the same quality work. Over a career, that's hundreds of citations attributable purely to having "Chen" instead of "Wojciechowski."
