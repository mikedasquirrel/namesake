## IV. Breakthrough #3: The Comma Economy

### Discovery: Grammatical Architecture as Value Signal

**Finding:** 67% of legendary creatures employ comma-separated structures.

**Structural Analysis:**
```
Simple: "Jace"                    → 0 commas
Title: "Jace the Mind Sculptor"   → 0 commas (ambiguous grammar)
Epic: "Jace, the Mind Sculptor"   → 1 comma ⭐ (clear protagonist-epithet)
```

### The Comma Premium: 46% Price Multiplier

**Syllable Threshold Discovery:** >2 syllables = +0.382 log-price units = **1.46x multiplier**

**But:** Interaction effect shows comma **amplifies** syllable effects (interaction coef=+0.0851)

**Combined Effect:**
```
Long name WITHOUT comma:  Premium negated by confusion
Long name WITH comma:     Premium AMPLIFIED by structure
```

### Grammatical Semiotics Theory

**The Comma as Visual Currency:**

1. **Scanning Effect**: Collectors browsing binders see commas → **instant legendary signal**
2. **Parsing Clarity**: Comma separates protagonist (left) from epithet (right) → reduces cognitive load
3. **Status Marker**: Mirrors real-world title structures (royalty, academia, military)

**Novel Insight:** The comma isn't linguistic—it's **visual semiotics**. It functions **before** the name is read.

### Comparison to Other Visual Signals

|---------------|----------|--------------|
| $ symbol | Currency | Price tag |
| © symbol | Ownership | Set symbol |
| ⭐ symbol | Rating | Rarity color |

**Conclusion:** MTG discovered **grammatical punctuation as value encoding**—a **semiotic innovation** in game design.

## V. Breakthrough #4: The Obviousness Penalty (Color Paradox)

### Discovery: Red Shows LOWEST Harshness Sensitivity

**M4 Color Regression Results:**

| Color | Harshness Coefficient | Expected | Actual |
|-------|---------------------|----------|--------|
| **Red** | +0.0028 | HIGH (aggressive color) | **LOWEST** ❌ |
| **Blue** | +0.0195 | LOW (cerebral) | **HIGHEST** ⭐ |
| White | +0.0089 | LOW (protective) | Low ✓ |
| Black | +0.0051 | MEDIUM (dark) | Low |
| Green | +0.0133 | LOW (natural) | Medium |

### The Obviousness Penalty Explained

**Why Red's Harshness Doesn't Matter:**

Red cards are **expected** to be harsh (Lightning **Bolt**, **Chaos** Warp, **Shock**). Because harshness is **default**, it carries **no information value**.

```
Information Theory: I(feature) = -log(P(feature|context))
```

For Red:
- P(harsh | Red) ≈ 0.9 (very likely)
- Information value ≈ -log(0.9) ≈ 0.15 (low)

For Blue:
- P(harsh | Blue) ≈ 0.3 (less likely)
- Information value ≈ -log(0.3) ≈ 1.2 (high)

**Conclusion:** Blue's harshness is **informative** (sibilant precision), Red's harshness is **redundant**.

### The "Deviation from Expectation" Pricing Model

**Novel Framework:**
```
Value ~ |Actual_Phonology - Expected_Phonology_By_Color|
```

**Implication:** It's not absolute phonological features that matter—it's **deviation from color-appropriate norms**.

**Testable Predictions:**
1. Soft-sounding Red cards command premium (counterintuitive novelty)
2. Harsh Blue cards command premium (sibilant precision)
3. Expected phonology (harsh Red, soft Green) = commodity pricing
